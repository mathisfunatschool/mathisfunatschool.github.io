Chromebookgames download guide

If you are using chromebookgames.github.io or one of our alias domains[1], you might want to download it. The download function makes it essentially unblockable for IT.

Below is a download guide, for choosing what you want to use.

You want to use the local file, because the about blank was one I added to be cool.

  1. Go to chromebookgames.github.io or an alias
  2. Find the download link at the bottom of the page. Chances are you have done this and are here
  3. Choose “LOCAL FILE” from the list and click on it.
  4. Open the files app on your chromebook.
  5. You will be able to find it in downloads, under the file name “CBGlocal.html”.
  6. Double click this file to run

Congratulations. You now have a file version of the website. It will still receive updates as normally, but is also unblockable. If I were you, I would bookmark this under something like “Google Sheets”.
